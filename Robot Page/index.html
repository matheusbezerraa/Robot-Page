Robot Bot

Bot do Telegram

O que ele faz?

1. Pergunta qual dais três opções você quer verificar. Linkedin, GitHub ou Currículo

2. Após escolher uma das três opções ele envia um link de uma delas.

3. Responder qualquer outra coisa não vai funcionar, e ele vai voltar para o início.
